# NetEquipment - Система учета сетевого оборудования

Полностью автономное веб-приложение для учета сетевого оборудования, работающее напрямую в браузере без установки сервера и баз данных.

## 🚀 Особенности

- **Полностью автономно** - Работает в любом современном браузере без установки
- **Без сервера** - Все данные хранятся в браузере (LocalStorage)
- **Excel интеграция** - Импорт/экспорт данных через Excel файлы
- **Современный интерфейс** - Bootstrap 5 с темной/светлой темой
- **Система пользователей** - Три уровня доступа с шифрованием паролей
- **QR-коды** - Автоматическая генерация для каждого оборудования
- **Поиск и фильтрация** - Мгновенный поиск и гибкая фильтрация
- **Статистика** - Красивые диаграммы и аналитика
- **Резервное копирование** - Автоматические бэкапы данных

## 📋 Требования

- Современный браузер (Chrome, Firefox, Edge, Safari)
- Интернет-соединение (только для загрузки CDN библиотек при первом запуске)

## 🗄️ Хранение данных

### LocalStorage
Все данные хранятся в LocalStorage браузера:

- `netequipment_users` - Пользователи
- `netequipment_equipment` - Оборудование
- `netequipment_logs` - Журнал действий
- `netequipment_settings` - Настройки
- `netequipment_current_user` - Текущий пользователь
- `netequipment_theme` - Тема оформления
- `netequipment_backups` - Резервные копии

### Excel файлы
- Основной способ обмена данными

## 🐛 Troubleshooting

### Данные не сохраняются
- Проверьте, что LocalStorage включен в браузере
- Убедитесь, что достаточно места в LocalStorage
- Попробуйте другой браузер

### Excel не импортируется
- Проверьте формат файла (.xlsx или .xls)
- Убедитесь, что первая строка содержит заголовки
- Проверьте обязательные поля
- Проверьте уникальность инвентарных номеров

### Пароль не подходит
- Используйте admin/admin123 для первого входа
- Если не работает, очистите LocalStorage и начните заново

### Интерфейс выглядит некорректно
- Обновите страницу (F5)
- Проверьте интернет-соединение (для CDN)
- Попробуйте другой браузер

### Очистка данных
Если нужно начать заново:
1. Откройте DevTools (F12)
2. Перейдите в Application → Local Storage
3. Удалите все ключи netequipment_*
4. Обновите страницу
